# OrangeHRM Recruitment & Vacancies Automation Testing

This project contains automated UI tests for the **Recruitment area**, specifically the **Vacancies module**, of the **OrangeHRM** system using **Selenium WebDriver** and **Pytest**.

## Testing Scope
- Recruitment module
- Vacancies management
- Vacancy creation and deletion
- Filtering and pagination validation
- UI behavior verification

## Tools Used
- Python
- Selenium WebDriver
- Pytest

## Project Structure

recruitment/ ├── test_recruitment.py ├── test_vacancies.py ├── report.html pytest.ini

## How to Run
Install dependencies:
```bash
pip install selenium pytest

Run all tests:

pytest -v

Run tests with HTML report:

pytest -v --html=recruitment/report.html

Test Target

Application: OrangeHRM

Area Tested: Recruitment

Module: Vacancies
